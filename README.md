Element.unselectable
===========
![Screenshot](http://marcelo.origoni.com.ar/images/Element.unselectable.png)
	- Element.unselectable add unselectable property to any given html elements.
	  
How to use
----------

	1.	#JS
		$('id').unselectable();
		$$('.class').unselectable();
		$$('#id').unselectable();
